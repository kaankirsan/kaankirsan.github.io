Robotics

Autonomous Solar Panel Cleaning Robot

Autonomous robot for solar panel cleaning with 85%+ autonomy achievement. Features depth cameras for obstacle detection, SLAM for navigation and environment mapping, 95%+ accuracy solar panel detection with YOLO, and human-following capability. Battery system with Smart BMS integration.

85% Autonomous Operation

Project Overview

What I Built

Autonomous navigation system using OAK-D depth cameras for real-time obstacle detection and avoidance. SLAM algorithms map the environment while the robot navigates. Computer vision pipeline with YOLOv5/v8 detects solar panels with 95%+ accuracy. Flutter mobile app provides real-time telemetry and remote control. Battery management system with Smart BMS handles power distribution and monitoring.

Key Achievements

  • 85%+ autonomous operation rate in field testing
  • 95%+ solar panel detection accuracy with YOLO
  • Real-time environment mapping with SLAM
  • Human-following capability for manual guidance
  • Mobile app for monitoring and control via Flutter
