Project Overview

What I Built

Implemented web and local UI using WinCC Unified HMI communicating directly with Siemens PLC via Profinet protocol. Integrated multiple high-precision sensors to measure jet fuel flow and pressure. Developed PID controls to regulate fuel flow and pressure using proportional valves, achieving 0.0001 L/min and 0.01 bar precision.

Implemented fire risk detection and protection systems. All sensors and control panels are explosion-proof certified for safe operation in hazardous environments.
